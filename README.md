# AI PDF Analyzer

A modern PDF analysis and summarization tool built with Python and Streamlit.

## Features

* Upload PDF documents
* Extract text from PDFs
* Generate automatic summaries
* Display document statistics
* Clean and interactive interface

## Technologies Used

* Python
* Streamlit
* PyPDF2

## Functionality

The application can:

* Read PDF files
* Extract readable text
* Analyze document content
* Generate readable summaries
* Show document metrics

## Run Locally

```bash id="b9l4vt"
pip install -r requirements.txt
streamlit run app.py
```

## Purpose

This project demonstrates document processing, text extraction and AI-assisted workflow development using Python.
